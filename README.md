# 🌍 Life-AI: Life Expectancy Prediction

**Life-AI** is a Machine Learning-powered web application designed to predict life expectancy based on global health and socioeconomic parameters. By leveraging a **Multiple Linear Regression** model, this project explores the relationships between factors like immunization, mortality, and economic status to provide data-driven insights.

---

## 🚀 Features
* **Predictive Modeling:** Uses a trained ML model to estimate life expectancy based on 19 different parameters.
* **Web Interface:** Built with **Flask**, allowing users to input data via a clean, intuitive web form.
* **Data-Driven Insights:** Analyzes factors such as GDP, Alcohol consumption, BMI, and Schooling.
* **Responsive UI:** Designed with HTML/CSS for a seamless user experience.

---

## 🛠️ Tech Stack
* **Language:** Python 🐍
* **ML Libraries:** NumPy, Pandas, Scikit-learn
* **Web Framework:** Flask
* **Database/Storage:** CSV (Training Data), Pickle (Model Serialization)
